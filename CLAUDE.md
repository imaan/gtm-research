# gtm-research

Research toolkit for analyzing go-to-market models.

## Skills

- `/research-init` — Set research context (run first)
- `/research-social` — Social channel analysis
- `/research-competitive` — Competitive landscape mapping
- `/research-playground` — (Optional) Generate interactive HTML dashboard from completed reports

## Key files

- `.claude/research-context.md` — Shared context file generated by /research-init, read by all other skills
- `reports/` — Where generated analysis reports are saved
- `input/` — Place source research documents here

## Conventions

- Always cite sources when making claims
- Use WebSearch and WebFetch for research
- Distinguish between hard data and inferences
- Note confidence levels on estimates
